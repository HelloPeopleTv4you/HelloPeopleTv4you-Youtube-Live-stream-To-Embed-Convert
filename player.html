How It Works

This tool allows you to embed a live YouTube stream on this page. Follow these steps:

  1. Enter YouTube Channel ID or Username: Type the channel ID or username of the YouTube channel you want to embed.
  2. Select Player Type: Choose whether you are using a channel ID and handle or just the username.
  3. Generate Embed: Click the "Generate Embed" button to load the live stream in the iframe below.
  4. View Live Stream: The live stream will be displayed in the iframe. If the iframe fails to load, a fallback image will be shown.

Embedding Methods

If your website is secured with an SSL certificate, use the HTTPS protocol for the embeddable URL. Ensure your streaming URL also supports the HTTPS format:

Example: https://HelloPeopleTv4you.github.io/m3u8-to-iframe/live/player1.html?url=https://example.com/playlist.m3u8

Example of Embed Iframe:

<iframe 
        frameborder="0" 
        height="100%" 
        width="450px" 
        allowfullscreen="" 
        scrolling="no" 
        allow="autoplay;fullscreen" 
        src="https://HelloPeopleTv4you.github.io/m3u8-to-iframe/live/player1.html?url=http://example.com/live.m3u8">
    </iframe>

HTTP Method (For Non-Secured Websites)

If your website does not have an SSL certificate, use the HTTPS protocol for embedding:

Example: https://HelloPeopleTv4you.github.io/m3u8-to-iframe/live/player1.html?url=https://example.com/playlist.m3u8
